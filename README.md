# front-end-learning
前端构建商品详情静态页面
学习前端最简单的方法就是找到能够实际练习的教程，并且自己手把手进行实践，完成作品增强信息
目前b站有很多优秀的视频素材去对照练习
开发软件也有像vscode这样的免费软件
只要我们理解了CSS的核心概念：padding、margin、border
以及常用组件：nav、section等
